PRODUCTION

This is 5 pieces made by the task of the class.

TASK １：デジタルキュビズム

processingを使えばいろんな形を直感的に表現することが出来るんだと学びました。絵を書くための関数がたくさん用意されていて、for文を使ったりすることで思いがけない図形が表現できたりして楽しかったです。今回の課題では円を用いて花火を表現しようと思いました。そのために二重for文で回してミルフィーユ状に線を重ねてみました。授業ページに載せていただいたので嬉しかったです。

View code

TASK ２：動画の再構成

初めて動画のファイルの扱い方を学びました。色情報を静止画と同じようにget(int x, int y)メソッドを用いて参照することができたので、改めて動画は静止画の集まりなんだなと認識しました。今回の作品も授業のページの作品ギャラリーに載せていただいて嬉しかったです。ただ、正直に言ってなぜこの作品がこの色合いを表現できたのかをわかっていません。直感でプログラムを組み合わせたら偶然できたものです。実際に動かしてみて思いもよらぬものが作れることもあるので積極的に動かしていこうと思いました。

View code

TASK ３：動作の再構成

モーションデータを利用して視覚化・可聴化する方法を学びました。視覚化・可聴化することによって、普通に聞いて動きを見るだけでは得ることができない視点から物事を見ることができるんだなと思いました。すごくアーティスティックな感じの作品を作ることができ、よくあるおしゃれな感じのやつはこんな風に作られているのかと理解しました。これから見るものもこんな視点から見ることによって新たな発見があると感じました。

View code

TASK ４：数理シミュレーション

糸掛け数楽アートを用いて数理シミュレーションについて学びました。プログラムを書く上では数学もやはり重要なんだなと感じました。一見するととても簡単なことをひたすらに繰り返していくと様々な形状を作り出すことができます。まさに「チリも積もれば山となる」と言った感じで、何事も真剣にコツコツと続けていけば努力は報われるんだなということを学びました。

View code

TASK ５：クリスマスイルミネーション

Tinkercadを使ってArduinoを疑似体験しました。パソコン上で動かしてみることができるので簡単に体験することができました。しかし、実機でやるのと比べてワクワク感だったり達成感が弱くなってしまうと感じました。だから、できれば実際に自分の手で配線を結んでやりたいなと思いました。作品は花火をイメージして下から順番に色がついていくようにし、三色の色を交互に点灯するように工夫しました。

Impressions

この授業は今期の中で一番実践的で、この先の何かに活かすことができると思いました。どうやって作るのか想像もつかなかった物をコードで表すことを学んだことによって、今まで漠然としか捉えることができなかった様々なデザインをプログラミングで表現することができました。 processingでプログラムを書きながら授業を進めることによって、何かを製作する時の思考手順を具体的に学ぶことができてとてもためになりました。だから、デザインだけではなく自分が何かを製作するときに助けになると思いました。この授業ではまだほんのさわりの部分しかやっていないと思うので、これからは自分なりにオリジナルの作品を作っていけるようにここで学んだことを活かしていきます。
